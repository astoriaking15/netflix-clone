# Netflix-Clone 🎬🍿
A responsive Netflix Landing Page clone focused on pixel-perfect UI recreation using semantic HTML5 and CSS3

## 🚀 Live Demo
[View the Live Project Here](https://astoriaking15.github.io/netflix-clone/)

## ✨ Features
- Responsive design for mobile and desktop.
- Hover effects on buttons and movie cards.
- Custom scrollbar styling.

# 🛠️ Technologies Used:
- HTML5
- CSS3 

# How to Run:
Simply clone the repository and open index.html in your browser.
